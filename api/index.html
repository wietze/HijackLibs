APIs

Datasets

Datasets of the {{ site.title }} project are available in the following formats:

Format Type Structure URL
JSON All entries Same as original YAML files {{'/api/hijacklibs.json' | relative_url}}
CSV All entries One line for each (DLL, EXE) combination {{'/api/hijacklibs.csv' | relative_url}}
YAML Individual entries The 'source' YAML files https://raw.githubusercontent.com/{{site.repo}}/main/yml

Detection content

Furthermore, there are also datasets for detection content:

Format Type Content URL
Individual Detections
σSigma (YAML) One rule for each DLL Auto-generated detection content for image loads {{'/api/sigma_feed_image.yml' | relative_url}}
σSigma (YAML) One rule for each DLL Auto-generated detection content for file writes {{'/api/sigma_feed_file.yml' | relative_url}}
σSigma (YAML) One rule for each DLL Auto-generated detection content for (unsigned) image loads {{'/api/sigma_feed_signature.yml' | relative_url}}
Single Detections
σSigma (YAML) A single rule for all Auto-generated detection content for file writes {{'/api/sigma_feed_file_single.yml' | relative_url}}
JSON A single file for all Auto-generated mapping between DLLs and their expected locations {{'/api/hijacklibs_single.json' | relative_url}}
